# 🕹️ Street Fighter (2026) - Landing Page

Esta é a landing page oficial (conceito) para o filme **Street Fighter (2026)**. O projeto foca em uma estética "Cyber-Arcade", misturando elementos táticos de HUD com o dinamismo visual da franquia.

## 🚀 Tecnologias e Design
- **Core**: HTML5 semântico e CSS3 modular.
- **Interatividade**: Vanilla JavaScript (zero dependências externas para máxima performance).
- **Estética**: Glassmorphism, Marquee dinâmico e animações baseadas em Intersection Observer.
- **Tipografia**: Orbitron (HUD/Títulos) e Inter (Leitura).

## 💎 Diferenciais
- **Performance**: Carregamento otimizado. O sistema de música e vídeos pesados foi removido para garantir que a experiência seja fluida em qualquer dispositivo.
- **UI/UX**: Cursor tático customizado, sistema de seleção de lutadores estilo arcade e carrossel de depoimentos que não interrompe a navegação.
- **Responsividade**: Layout adaptável para smartphones, tablets e desktops de alta resolução.

## 📁 Estrutura de Arquivos
- `index.html`: estrutura do site e SVGs otimizados.
- `style.css`: sistema de design, animações e tokens de cores.
- `script.js`: lógica de interface, seleção de lutadores e sistemas de scroll.
- `image/`: assets visuais e renders de alta qualidade.

## 🛠️ Como Visualizar
Basta abrir o arquivo `index.html` em qualquer navegador moderno. Não requer servidor backend ou instalação de pacotes.
